# direito.love — Chat PWA (fluxo fixo)

Fluxo fixo implementado:
1) Saudação inteligente (inclui “Noooite, acordado ainda?” entre 01–05h)
2) “Vamos lá, vou criar um prompt top pra você.”
3) “Qual o tema? *Ex.: Danos estéticos relacionados a acidentes de trabalho*”
→ usuário envia
4) “Legal, me dá um segundo, já te entrego o prompt pronto.”
5) “Pronto, é só clicar para copiar e depois colar na sua I.A. preferida.”
→ 3 botões (ChatGPT/Gemini/Perplexity)
→ Modal (Ir para IA / Fechar), copia no clique
→ Ao fechar modal: “✅ Prompt copiado com sucesso!” + botão “Nova pesquisa” dentro do chat

Outros:
- Botões flutuantes da base: REMOVIDOS
- Aba “Início” reinicia a pesquisa
- “Arquivo” abre histórico e copiar usa o mesmo modal
