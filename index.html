    ❓ Como Usar

    1. Digite um tema em até 80 caracteres.

    2. Escolha os elementos que deseja incluir no prompt.

    3. Opcional: selecione conteúdos jurídicos sugeridos.

    4. Gere o prompt e copie, salve ou compartilhe.

    ⚠️ Lembre-se: este app é apenas para fins educacionais.

    direito.love é um app educacional para estudantes de Direito.
